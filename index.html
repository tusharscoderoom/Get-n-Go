Get-n-Go

Press Any Key to Start the Game!

Rules are Simple ⬇️

  • #1 : You can use any button to start the game.
  • #2 : Each colored tile(button) has some sound attached to it.
  • #3 : You will hear a sound showing you the button press that make the sound.
  • #4 : Now you have to press(touch) tile in the same sequence shown to you from the starting.
  • #5 : Each round will add one new tile.
  • #6 : Follow the sequence and sharpen your memory.

Enjoy :)
